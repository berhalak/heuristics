# Look for a platform that allows customisations

## Question
What if I need a customisation on a Commercial-of-the-shelf (COTS)?

## Heuristic
Look for a platform that allows customisations.

## Examples
- AWS Cognito allows hooks for custom behaviour on events
- Webhooks standard
